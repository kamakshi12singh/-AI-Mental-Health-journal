# AI Mental Health Companion

A beginner-friendly MVP of an AI Mental Health Companion web application.

## Features
- **Diary Input**: Write your thoughts and feelings.
- **Mood Analysis**: Uses VADER sentiment analysis to categorize your mood.
- **Rule-based Advice**: Get simple advice based on your mood.
- **Mood Trends**: Visualize your mood over the last week.
- **Streak Tracking**: Keep track of your journaling habits.

## Tech Stack
- **Frontend/Backend**: Streamlit
- **Analysis**: vaderSentiment
- **Database**: SQLite
- **Visualization**: Pandas

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```
